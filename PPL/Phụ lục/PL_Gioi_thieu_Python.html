Mở đầu

CO3005 · Phụ lục: Giới thiệu về Ngôn ngữ Lập trình Python · Trần Ngọc Bảo Duy

Python là một trong những ngôn ngữ lập trình phổ biến nhất hiện nay, được sử dụng rộng rãi trong giảng dạy lập trình, phát triển phần mềm, phân tích dữ liệu, trí tuệ nhân tạo, tự động hóa và xây dựng hệ thống web. Trong bối cảnh môn học Nguyên lý ngôn ngữ lập trình, Python không chỉ là một ngôn ngữ thực hành thuận tiện, mà còn là một ví dụ tốt để quan sát nhiều khái niệm quan trọng như biến (variable), phạm vi tĩnh (static scoping), kiểu động (dynamic typing), ngăn xếp lời gọi (call stack), đệ quy (recursion), câu lệnh điều kiện (conditional statement), danh sách (list), bộ (tuple), từ điển (dictionary), vòng lặp (loop) và hàm (function).

Phụ lục này giới thiệu Python ở mức nền tảng, nhằm hỗ trợ người học đọc, hiểu và viết được các chương trình Python đơn giản. Trọng tâm không phải là trình bày toàn bộ ngôn ngữ Python, mà là nắm các đặc điểm cốt lõi có liên quan trực tiếp đến việc học nguyên lý ngôn ngữ lập trình.

Vì sao chọn Python?

CO3005 · Phụ lục: Giới thiệu về Ngôn ngữ Lập trình Python

Python được tạo ra bởi Guido van Rossum và xuất hiện lần đầu vào năm 1991. Từ đó đến nay, Python phát triển thành một ngôn ngữ lập trình được sử dụng rộng rãi trong nhiều lĩnh vực khác nhau. Một lý do quan trọng khiến Python phổ biến là ngôn ngữ này được thiết kế với mục tiêu dễ đọc, dễ viết và gần với cách con người diễn đạt thuật toán.

Python
message = "Hello, Python!"
print(message)

So với nhiều ngôn ngữ khác, Python không yêu cầu khai báo kiểu dữ liệu trước, không cần dấu chấm phẩy ở cuối câu lệnh và không cần hàm main trong những chương trình nhỏ. Ví dụ, cùng một ý tưởng in ra màn hình, trong C ta thường viết:

C
#include <stdio.h>
int main() {
    printf("Hello, C!\n");
    return 0;
}

Trong khi đó, Python chỉ cần:

Python
print("Hello, Python!")

Điều này không có nghĩa Python luôn tốt hơn C. C phù hợp với các bài toán cần kiểm soát bộ nhớ, hiệu năng và phần cứng ở mức thấp. Python phù hợp hơn khi người lập trình cần viết nhanh, đọc dễ, thử nghiệm thuận tiện và tận dụng hệ sinh thái thư viện phong phú.

Python được sử dụng trong nhiều miền ứng dụng. Trong lập trình web, Python có các thư viện và framework như Django, Flask hoặc FastAPI. Trong khoa học dữ liệu, Python có NumPy, pandas, matplotlib và scikit-learn. Trong trí tuệ nhân tạo, Python có PyTorch, TensorFlow và nhiều công cụ học máy khác.

Python – Xử lý danh sách điểm
scores = [8.0, 7.5, 9.0, 6.5]
average = sum(scores) / len(scores)
print("Average score:", average)

# Lọc điểm đạt
passed_scores = []
for score in scores:
    if score >= 7.0:
        passed_scores.append(score)
print(passed_scores)

Một đặc điểm nổi bật của Python là khả năng đọc hiểu (readability). Python sử dụng xuống dòng để kết thúc câu lệnh thông thường và sử dụng thụt lề (indentation) để xác định phạm vi của khối lệnh. Python không dùng cặp dấu ngoặc nhọn { } như C, C++ hoặc Java để đánh dấu khối lệnh:

Python – Thụt lề xác định khối lệnh
x = 10
if x > 0:
    print("positive")
    print("done")

Python sử dụng phạm vi tĩnh (static scoping hoặc lexical scoping). Python cũng là ngôn ngữ kiểu động (dynamically typed), nghĩa là kiểu của giá trị được kiểm tra trong quá trình chạy, còn tên biến không cần được khai báo kiểu trước:

Python – Kiểu động
x = 10
print(type(x))   # <class 'int'>
x = "hello"
print(type(x))   # <class 'str'>

Mô hình thực thi

CO3005 · Phụ lục: Giới thiệu về Ngôn ngữ Lập trình Python

Để hiểu Python, ta cần hiểu chương trình Python được thực thi theo trình tự như thế nào. Ở mức đơn giản, chương trình Python được đọc từ trên xuống dưới, và các câu lệnh ở mức ngoài cùng được thực thi theo thứ tự xuất hiện. Khi gặp định nghĩa hàm, Python tạo ra một đối tượng hàm và gắn nó với tên hàm, nhưng phần thân hàm chưa được thực thi ngay.

Python – Thứ tự thực thi
c = 3
def gt(n):
    if n == 0 or n == 1:
        return 1
    else:
        return n * gt(n - 1)
print(gt(c))
def foo(a):
    return a * c
print(foo(2))

Đệ quy và call stack

Hàm gt là một ví dụ của hàm đệ quy (recursive function), tức là hàm có lời gọi đến chính nó. Với gt(3), quá trình tính toán có thể hiểu như sau:

Truy vết đệ quy
gt(3)
= 3 * gt(2)
= 3 * 2 * gt(1)
= 3 * 2 * 1
= 6

Khi một hàm được gọi, Python tạo ra một khung lời gọi (stack frame) để lưu các thông tin của lần gọi đó. Các khung lời gọi được tổ chức trong ngăn xếp lời gọi (call stack):

Call stack – gt(3)
gt(n = 1)  -> returns 1
gt(n = 2)  -> waiting for result of gt(1)
gt(n = 3)  -> waiting for result of gt(2)
main       -> calls gt(c)
Khi viết hàm đệ quy, luôn cần có điều kiện dừng (base case). Nếu không, Python sẽ gọi hàm liên tục và gây lỗi RecursionError: maximum recursion depth exceeded.

Phạm vi tĩnh

Trong Python, biến ở phạm vi ngoài có thể được đọc bên trong hàm nếu không bị che khuất bởi biến cục bộ cùng tên:

Python – Phạm vi tĩnh
c = 3
def foo(a):
    return a * c    # c được tìm ở phạm vi ngoài
print(foo(2))  # 6

# Nếu có phép gán bên trong, Python xem là biến cục bộ
def bar():
    c = 10       # biến cục bộ, khác c toàn cục
    return c
print(bar())  # 10
print(c)       # 3 – c toàn cục không đổi

Biến và câu lệnh

CO3005 · Phụ lục: Giới thiệu về Ngôn ngữ Lập trình Python

Quy tắc đặt tên

Trong Python, biến (variable) là một tên dùng để tham chiếu đến một đối tượng. Một tên biến trong Python được gắn (bind) với một đối tượng chứ không phải là một ô nhớ cố định.

Python – Kiểu động
x = 23
print(type(x))  # <class 'int'>
x = "Python"
print(type(x))  # <class 'str'>

Một định danh có thể gồm chữ cái, chữ số và dấu gạch dưới _, nhưng không được bắt đầu bằng chữ số. Python phân biệt chữ hoa và chữ thường (case-sensitive):

Python – Tên hợp lệ và không hợp lệ
# Hợp lệ
count = 10
student_name = "An"
_total = 100

# Không hợp lệ
# 1x = 5        # bắt đầu bằng chữ số
# student-name  # dấu - bị coi là phép trừ

# Case-sensitive
count = 1; Count = 2; COUNT = 3
print(count, Count, COUNT)  # 1 2 3

Chú thích

Chú thích (comment) là phần văn bản trong mã nguồn dùng để giải thích chương trình cho con người. Trình thông dịch bỏ qua chú thích khi thực thi. Trong Python, chú thích một dòng bắt đầu bằng ký tự #:

Python – Chú thích và docstring
# This is a line comment
x = 23  # This is an inline comment

def square(x):
    """
    Return the square of x.
    """
    return x * x

help(square)   # docstring được công cụ đọc tài liệu sử dụng

Phép gán và câu lệnh rỗng

Phép gán (assignment) trong Python dùng để gắn một tên với một giá trị hoặc một đối tượng. Python hỗ trợ nhiều dạng phép gán tiện lợi:

Python – Các dạng phép gán
# Hoán đổi không cần biến tạm
a = 10; b = 20
a, b = b, a
print(a, b)   # 20 10

# Gán nhiều tên cùng một giá trị
a = b = c = 23

# Câu lệnh rỗng – dùng khi cần khối lệnh hợp lệ
x = -1
if x < 0:
    pass

Câu lệnh điều kiện

Câu lệnh điều kiện (if statement) cho phép chương trình lựa chọn nhánh thực thi dựa trên giá trị đúng hoặc sai của một điều kiện:

Python – if / elif / else
score = 8.5
if score >= 9.0:
    print("excellent")
elif score >= 7.0:
    print("good")
elif score >= 5.0:
    print("average")
else:
    print("failed")

Một ưu điểm của cách dùng thụt lề là Python tránh được lỗi else treo (dangling else) thường được thảo luận trong các ngôn ngữ có cú pháp khối khác. Trong Python, else thuộc về if có cùng mức thụt lề gần nhất, nên cấu trúc rõ ràng hơn.

Cấu trúc dữ liệu cơ bản

CO3005 · Phụ lục: Giới thiệu về Ngôn ngữ Lập trình Python

List

Danh sách (list) là dãy các phần tử có thứ tự và có thể thay đổi. Danh sách thường được tạo bằng cặp ngoặc vuông [].

Python – List
numbers = [10, 20, 30]
print(numbers[0])   # 10 – phần tử đầu tiên
print(numbers[-1])  # 30 – phần tử cuối cùng
print(len(numbers))  # 3

items = ["a", "b"]
items.append("c")     # thêm phần tử vào cuối
print(items)          # ['a', 'b', 'c']

mixed = [1, "Python", 3.14, True]  # kiểu hỗn hợp

Tuple

Bộ (tuple) cũng là dãy có thứ tự, nhưng không thể thay đổi sau khi tạo. Tuple thường được tạo bằng cặp ngoặc tròn ().

Python – Tuple
point = (3, 4)
print(point[0], point[1])

# Tuple một phần tử – cần dấu phẩy
t = (42,)
print(type(t))   # <class 'tuple'>

# Unpacking
a, b = 10, 20
a, b = b, a
print(a, b)   # 20 10

# Hàm trả về nhiều giá trị qua tuple
def divmod_pair(a, b):
    return a // b, a % b
q, r = divmod_pair(17, 5)
print(q, r)   # 3 2

# Tuple có thể làm khóa dictionary (vì bất biến)
grid = {(0, 0): "origin", (1, 2): "point"}
print(grid[(1, 2)])

Dictionary

Từ điển (dictionary) biểu diễn ánh xạ từ khóa (key) sang giá trị (value). Từ điển thường được tạo bằng cặp ngoặc nhọn {}.

Python – Dictionary
student = {"id": "SV01", "name": "An", "score": 8.0}
print(student["name"])    # An

# Cập nhật và thêm khóa
student["score"] = 8.5
student["passed"] = True

# Kiểm tra khóa và get an toàn
print("score" in student)            # True
print(student.get("grade", "N/A"))   # N/A

# Duyệt từ điển
for key, value in student.items():
    print(key, "->", value)

# Danh sách các dictionary
students = [
    {"id": "SV01", "name": "An", "score": 8.0},
    {"id": "SV02", "name": "Binh", "score": 4.5},
]
for s in students:
    print(s["id"], s["name"], s["score"])
Khóa trong dictionary phải là đối tượng hashable, chẳng hạn str, int hoặc tuple. List không thể làm khóa vì nó thay đổi được.

Vòng lặp và hàm

Vòng lặp while

Vòng lặp while lặp lại một khối lệnh khi điều kiện còn đúng. Phần else (tùy chọn) được thực hiện khi vòng lặp kết thúc bình thường (không bị dừng bởi break):

Python – while với else
i = 1
while i <= 5:
    print(i)
    i = i + 1
else:
    print("done")   # in ra vì điều kiện trở thành sai

# break ngăn else chạy
i = 1
while i <= 5:
    if i == 3:
        break
    print(i)
    i = i + 1
else:
    print("done")   # KHÔNG in ra

Vòng lặp for

Vòng lặp for dùng để duyệt qua các phần tử của một dãy hoặc một đối tượng có thể lặp. Hàm range thường được dùng để tạo dãy số:

Python – for
for x in [1, 2, 3]:
    print(x)

# range(1, 6) sinh 1..5 (không bao gồm 6)
for i in range(1, 6):
    print(i)

# Duyệt chuỗi
for ch in "abc":
    print(ch)

# Duyệt danh sách
names = ["An", "Binh", "Cuong"]
for name in names:
    print("Hello", name)

break, continue và else trong vòng lặp

Câu lệnh break kết thúc vòng lặp ngay lập tức. Câu lệnh continue bỏ qua phần còn lại của thân vòng lặp và chuyển sang lần lặp tiếp theo. Một ứng dụng phổ biến của for...else là tìm kiếm:

Python – for...else tìm kiếm
numbers = [2, 4, 6, 8]
target = 5
for x in numbers:
    if x == target:
        print("found")
        break
else:
    print("not found")   # in ra vì không có break

# continue
for x in range(1, 6):
    if x == 3:
        continue   # bỏ qua x = 3
    print(x)   # 1 2 4 5

Hàm

Hàm (function) là một đơn vị chương trình có tên, có thể nhận tham số, thực hiện một công việc và trả về kết quả. Trong Python, hàm được định nghĩa bằng từ khóa def:

Python – Hàm và đệ quy
def square(x):
    return x * x
print(square(5))  # 25

def is_even(n):
    return n % 2 == 0

# Hàm đệ quy – giải thừa
def giaithua(n):
    if n == 0 or n == 1:
        return 1
    return n * giaithua(n - 1)
print(giaithua(5))  # 120
Khi viết hàm đệ quy, luôn cần có điều kiện dừng. Nếu không, chương trình sẽ gọi hàm liên tục và gây lỗi vượt quá giới hạn đệ quy.

Sử dụng module

Python cho phép sử dụng lại mã có sẵn thông qua module. Để sử dụng module, ta dùng câu lệnh import:

Python – import module
import math
print(math.sqrt(16))      # 4.0
print(math.factorial(5)) # 120

from math import sqrt
print(sqrt(25))            # 5.0

import functools
result = functools.reduce(lambda x, y: x + y, [1, 2, 3])
print(result)               # 6

Tóm tắt phụ lục

CO3005 · Phụ lục: Giới thiệu về Ngôn ngữ Lập trình Python

Phụ lục này đã giới thiệu những đặc điểm nền tảng của Python dưới góc nhìn của môn Nguyên lý ngôn ngữ lập trình.

  • Python là ngôn ngữ phổ biến, dễ đọc, có cú pháp gọn và dùng thụt lề để xác định khối lệnh.
  • Python dùng phạm vi tĩnh và kiểm tra kiểu động trong quá trình chạy.
  • Biến trong Python là tên tham chiếu đến đối tượng, được tạo ra thông qua phép gán thay vì khai báo kiểu trước.
  • Các câu lệnh cơ bản như gán, điều kiện, vòng lặp và định nghĩa hàm tạo nên nền tảng để viết các chương trình Python đơn giản.
  • Python cung cấp các cấu trúc dữ liệu cơ bản như list, tuple và dictionary để biểu diễn dãy có thứ tự, nhóm giá trị bất biến và ánh xạ khóa–giá trị.
  • Python hỗ trợ đệ quy, tổ chức lời gọi hàm bằng ngăn xếp lời gọi, cung cấp vòng lặp while, vòng lặp for, cơ chế break, continue, else trong vòng lặp, và khả năng tái sử dụng mã thông qua module.

Những đặc điểm này giúp Python trở thành một ngôn ngữ phù hợp để minh họa nhiều khái niệm quan trọng trong nguyên lý ngôn ngữ lập trình.
